Browser Phishing Protection - Privacy Consent

The goal of this web extension is to enhance your online security by detecting phishing webpages. The extension is currently in a test mode. As such, the extension will run silently (i.e., no detection notifications and website blocks) by selectively collecting and analyzing visited URLs and features related to its visual appearance.

These indicators are only stored for webpages that exhibit characteristics suggestive of potential phishing threats and for which we could not yet determine their nature (i.e., benign or malicious).

The data empowers us to proactively safeguard against phishing attacks and the impersonation of genuine websites, ensuring your online experience remains safe and secure.

For the functionality of the extension, we collect the following categories of data (only for webpages that exhibit characteristics suggestive of potential phishing threats and for which we could not yet determine their nature):

Data Points
Purpose of Processing Your Personal Data
Security Data may include:
  • URL
  • Visual appearance
  • URL: The URL that was analyzed. This information allows us to verify whether we triggered a false positive.
  • Visual appearance: On the client, we record a mini thumbnail (224x224 px) of the page visible in the browser. We record float values representing the color of each pixel for prediction reproducibility and verification. The collected features do not enable us to read any person
Service Data may include:
  • Browser Installation ID
  • Execution time
  • Browser and OS version
  • Browser resolution (e.g., 1920 x 1080 px)
  • System memory
  • Graphics compatibility
  • Extension status
  • Service Data: Service data allow us to estimate the overhead that our extension introduces to optimize it based on different host properties.

Retention Period: All the data above is stored when submitted and deleted after 30 days.

Read more about the processing of data and your rights in our Global Privacy Statement.
